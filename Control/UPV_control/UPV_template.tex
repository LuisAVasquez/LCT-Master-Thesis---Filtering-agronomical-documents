\subsection{Second level title}
\label{sec:second-level-title}

To reference other works, use the following style: \citep{carleta96}. If
there are two authors, use \citep{ersan94}; for more than two authors, use
\citep{allen00}. For referencing more than one work, use: \citep{carleta96,
  ersan94, allen00}.

If the paper or work is the subject of a sentence, use this style:
\citet{allen00}.  For instance:''As noted by \citet{allen00}, ...''.

To quote sentences or paragraphs, use the following style:
paragrafoan:
\begin{quotation}
  Here comes the quote.
\end{quotation}
